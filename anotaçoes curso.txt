Git/GitHub

Branch
Branches são separações de código. O branch padrão do projeto é o master. Branches são usados para separar alterações grandes ou novas funcionalidades do projeto.

Criar uma branch: git branch [nome da branch]
Trocar de branch: git checkout [branch de destino]
